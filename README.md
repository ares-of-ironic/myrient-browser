# Myrient Browser

A fast CLI/TUI tool for browsing and downloading files from public HTTP repositories using a local path index. Built for [myrient.erista.me](https://myrient.erista.me) but works with any HTTP directory.

## ⚠️ Disclaimer

This tool is intended solely for downloading content you have legal rights to access. Ensure you comply with all applicable laws and terms of service.

## Features

- **Interactive TUI** — directory tree browser, fuzzy search, multi-selection
- **Fast search** — powered by [ripgrep](https://github.com/BurntSushi/ripgrep) (sub-second even for rare terms), OR operator (`|`) supported
- **File & folder sizes** — displayed in browser, info panel and download queue (requires JSON index)
- **Disk usage** — live `du -s downloads/` shown in the stats panel and screensaver
- **Segmented downloads** — parallel HTTP Range segments per file for maximum throughput
- **Dynamic concurrency** — adjust simultaneous downloads with `+`/`-` in the Downloads tab
- **Download queue** — parallel downloads with resume (HTTP Range), exponential backoff retry
- **Large directory support** — paginated listing (500 items/page) so folders with 15k+ files never freeze the UI
- **Queue management** — retry, priority promotion, remove, clear completed/failed, search and filter by status
- **Already-downloaded detection** — files already on disk are flagged when queued; forced re-download available
- **Screensaver** — Matrix rain background with live download stats and easter eggs (`~` or `` ` ``)
- **Export** — save selected paths to file (plain paths / URLs / JSON)
- **Persistent queue** — state saved to disk, survives restarts
- **Batch CLI mode** — scriptable commands for search, export, queue and download

## Requirements

- Python 3.11+
- macOS or Linux
- [`rg`](https://github.com/BurntSushi/ripgrep) — strongly recommended for fast search (falls back to Python if absent)
- [`rclone`](https://rclone.org) — required only for generating the index file

## Index Setup

The browser needs a local index of the remote repository. The recommended format is JSON (generated by `rclone lsjson`) because it includes file sizes. A plain text format is also supported.

### Generate with rclone (recommended)

```bash
# One-time rclone configuration
rclone config create Myrient http url https://myrient.erista.me/files/

# JSON index — includes file/folder sizes (~72 MB for myrient)
rclone lsjson --recursive Myrient: > directory/all_paths.json

# Plain text index — paths only, no sizes (~500 MB for myrient)
rclone lsf --recursive Myrient: > directory/all_paths.txt
```

A small sample index (`directory/sample_index.json`) is included for testing.

## Configuration

```bash
cp config.example.toml config.toml
```

Key settings in `config.toml`:

```toml
[server]
base_url = "https://myrient.erista.me/files"

[download]
concurrency = 4             # parallel downloads (adjustable at runtime with +/-)
retries = 3
segments_per_file = 4       # parallel HTTP Range segments per file (1 = off)
min_segmented_mb = 8.0      # minimum file size (MB) to trigger segmented download

[index]
index_file = "directory/all_paths.json"   # or all_paths.txt
search_limit = 500          # max results returned by fuzzy/rg search

[display]
use_decimal_units = true    # true = KB/MB/GB/TB  |  false = KiB/MiB/GiB/TiB
du_human_readable = false   # true = let du format output (e.g. "4.2G")
```

Environment variables are also supported:

```bash
export MYRIENT_BASE_URL="https://example.com/files"
export MYRIENT_CONCURRENCY=8
```

## Usage

### Interactive TUI

```bash
myrient
```

The index is loaded in the background — a loading screen is shown on startup.

#### Browser tab — keyboard shortcuts

| Key | Action |
|-----|--------|
| `/` | Focus search input |
| `Enter` | Enter directory |
| `Backspace` | Go up one level |
| `g` | Go to the folder containing the highlighted item (useful after search) |
| `Space` | Toggle selection (recursive for directories) |
| `a` | Select all visible items |
| `c` | Clear all selections |
| `d` | Download highlighted / selected items |
| `e` | Export highlighted / selected items |
| `m` | Toggle "show missing only" filter |
| `r` | Reload index |
| `]` | Next page (large directories) |
| `[` | Previous page (large directories) |
| `Escape` | Clear search / go to root |
| `h` | Help screen |
| `q` | Quit |

#### Downloads tab — keyboard shortcuts

| Key | Action |
|-----|--------|
| `/` | Focus search input |
| `1`–`5` | Filter: All / Queued / Active / Done / Failed |
| `p` | Retry / resume selected download (moves to front of queue) |
| `u` | Promote selected queued item to top priority |
| `x` | Remove selected from queue |
| `f` | Retry all failed downloads |
| `F` | Force re-download (even if file already exists on disk) |
| `k` | Clear completed downloads |
| `X` | Clear entire queue (confirmation required) |
| `+` / `-` | Increase / decrease concurrent download slots |
| `]` / `[` | Next / previous page |
| `Escape` | Clear search and filters |

#### Global shortcuts

| Key | Action |
|-----|--------|
| `~` / `` ` `` | Toggle screensaver (Matrix rain + live stats) |
| `h` | Help screen |
| `q` | Quit |

### Batch CLI

#### Search

```bash
myrient search "commodore"
myrient search "c64|commodore|amiga"   # OR search
myrient search "pacman" --print-urls
myrient search "bios" --files-only
myrient search "Nintendo" --dirs-only
```

#### Export

```bash
myrient export "c64" --out selection.txt          # plain paths
myrient export "c64" --out selection.txt --urls   # full URLs
myrient export "c64" --out selection.json --json  # JSON
myrient export "c64" --dry-run                    # preview only
```

#### Queue

```bash
myrient queue --from-selection selection.txt
myrient queue --paths "MAME/ROMs/pacman.zip" --paths "MAME/ROMs/galaga.zip"
myrient queue --from-selection selection.txt --dry-run
```

#### Download

```bash
myrient download --all-queued     # start all queued downloads
myrient download --retry-failed   # retry failed downloads
myrient download --status         # show queue status
```

#### Status

```bash
myrient status
```

## Export Formats

**Plain paths** (default)
```
MAME/ROMs/pacman.zip
MAME/ROMs/galaga.zip
```

**URLs**
```
https://myrient.erista.me/files/MAME/ROMs/pacman.zip
https://myrient.erista.me/files/MAME/ROMs/galaga.zip
```

**JSON**
```json
[
  {
    "path": "MAME/ROMs/pacman.zip",
    "url": "https://myrient.erista.me/files/MAME/ROMs/pacman.zip",
    "is_dir": false,
    "expanded_from_dir": "MAME/ROMs",
    "local_target": "downloads/MAME/ROMs/pacman.zip"
  }
]
```

## Security

- Path traversal validation on all user-supplied paths
- No "download all" shortcut — explicit selection required
- Confirmation dialog for large downloads (>1 GB or >10 files)
- Configurable rate limiting
- Operation logs in `logs/app.log`
